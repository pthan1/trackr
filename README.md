## Install and Set Up

To run the application:

You can see this application on the following link:

[Deployed Link](https://trackr-turing.herokuapp.com/)

To run Cypress tests:

1. Clone down the repo
2. Cd into `trackr`
3. Run `npm install`
4. Run `npm start`
5. Go to localhost:3000
